•  Pala: Para cavar hoyos, mover tierra o plantar árboles y arbustos.
•  Azadón: Ideal para remover la tierra y eliminar maleza.
•  Rastrillo: Para nivelar el suelo o recoger hojas y escombros.
•  Tijeras de podar: Para cortar ramas pequeñas y mantener las plantas podadas.
•  Manguera: Para regar plantas y mantener el jardín hidratado.
•  Regadera: Ideal para regar plantas más pequeñas o macetas.
•  Carretilla: Para transportar tierra, abono o plantas.
•  Desbrozadora: Para cortar hierba alta y maleza.
•  Cortacésped: Para mantener el césped corto y ordenado.
•  Serrucho de podar: Para cortar ramas más gruesas que no pueden cortarse con tijeras de podar.
•  Soplador de hojas: Para limpiar las hojas caídas y escombros de áreas grandes.
•  Sierra de pértiga: Para podar ramas altas sin necesidad de una escalera.
•  Malla protectora para plantas: Para proteger las plantas de plagas o animales.
